openmmslicer.moves module
=========================

Here one can define a range of moves which can be used for enhanced sampling:
:class:`~openmmslicer.moves.TranslationMove`, :class:`~openmmslicer.moves.RotationMove` and
:class:`~openmmslicer.moves.DihedralMove`, and combine them into a :class:`~openmmslicer.moves.MoveList`.
